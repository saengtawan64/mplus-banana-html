หน้าสรุปยอดขายสาขาแบบทดลอง

เว็บ static สำหรับดูภาพรวมยอดขาย ตรวจหัวคอลัมน์ CSV ดูสาขา และคัดลอกสรุปยอดขายแบบง่าย

เว็บ staticไม่มี backend หรือระบบ build
เตรียมไว้สำหรับ CSVยังไม่เชื่อม CSV จริง
Role ปลอดภัยใช้ role key ที่อนุมัติแล้วเท่านั้น
ดูยอดขาย ตรวจหัวคอลัมน์ CSV

มุมมองทดลอง

ตัวเลือกมุมมองทดลองเท่านั้น ไม่ใช่ระบบล็อกอินจริง และไม่ใช่ระบบกำหนดสิทธิ์จริง

โปรเจกต์แยกเดี่ยว

ไม่มี React, Vite, package file, Firebase writes หรือการพึ่งพาโปรเจกต์เก่า

เตรียมไว้สำหรับ CSV

เมื่อได้รับลิงก์ Published CSV จริง ระบบจะตรวจจากหัวคอลัมน์ ไม่ใช้ตำแหน่งคอลัมน์ตายตัว

Role key คงเดิม

ชื่อที่แสดงต้อง map กลับไปที่ CREATOR, OWNER, ADMIN, STAFF หรือ VIEWER เท่านั้น

สาขา

ดูชื่อสาขาภาษาไทยและสถานะการแสดงผล

เปรียบเทียบ

ดูตัวอย่างกราฟเปรียบเทียบก่อนเชื่อม CSV จริง

คัดลอกสรุป

เตรียมข้อความสำหรับคัดลอกเท่านั้น ไม่มี PDF หรือ Excel
