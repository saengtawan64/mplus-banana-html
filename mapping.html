Readiness boundary

Mapping is readiness-only

หน้านี้อธิบายกฎการจับคู่และความพร้อมเท่านั้น ยังไม่มี production CSV import

No production import

ยังไม่เปิดใช้งานการนำเข้า CSV สำหรับ production และไม่เพิ่ม live CSV URL ใหม่ในรอบนี้

Local guard checked

Parser มี local readiness guard 52/52 checks แต่ production integration ยังต้องอนุมัติแยก

Official formula

ยอดขายรวมทางการ = ยอดในระบบ + ยอดนอกระบบ / CSV ยอดรวมใช้ cross-check เท่านั้น

Missing is not zero

ค่าว่าง / - / — ต้องคงเป็น missing หรือ not applicable ไม่ใช่ 0
